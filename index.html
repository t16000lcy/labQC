AI Assistant

AI 小助理

小助理會先從 FAQ 與檢驗項目資料找答案。正式版可將問題送到後端，由 OpenAI API 搭配 RAG 知識庫回覆。

  • 可答：抽血時間、空腹、採檢注意事項、檢驗項目意義。
  • 不答：診斷、治療、用藥調整、個人報告嚴重度判斷。
  • 急症或個人病情請立即洽醫師或急診。

部署與維護規劃

資料維護

以 Google Sheet 維護檢驗項目，發布為 CSV 後填入 `app.js` 的 `GOOGLE_SHEET_CSV_URL`。

正式部署

前端可部署到 Vercel、Cloudflare Pages 或 Google Cloud Storage；AI 後端建議使用 Vercel Functions、Cloud Run 或 Cloud Functions。

網域與 SSL

網域可由 GoDaddy、Cloudflare 或 Namecheap 註冊，DNS 與 SSL 建議交給 Cloudflare 管理。
